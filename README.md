# Profile

A personal portfolio website built with Next.js, showcasing my background, skills, projects, and certifications.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Shadcn UI

## Getting Started

### Prerequisites

- Node.js 18+ or Bun
- Package manager: npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/shio265/profile.git
cd profile

# Install dependencies
npm install
# or
bun install
```

### Enviroment

- Rename ``.env.example`` to ``.env``
- Fill variable

> [!NOTE]  
> [Lanyard](https://github.com/Phineas/lanyard) has a different structure from [Lantern](https://github.com/discordplace/lantern). Iâ€™m using Lantern in this case.

### Development

Run the development server:

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build

Build the application for production:

```bash
npm run build
# or
bun run build
```

### Production

Start the production server:

```bash
npm start
# or
bun start
```

## Project Structure

```strucs
app/           # Next.js app directory
components/    # React components
  tabs/        # Tab section components
  ui/          # UI components
hooks/         # Custom React hooks
lib/           # Utility functions
public/        # Static assets
```
